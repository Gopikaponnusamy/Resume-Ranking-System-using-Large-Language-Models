Automated Resume Ranking System using Large Language Models
Overview:
This project implements an Automated Resume Ranking System using a Large Language Model (LLM) that analyzes candidate resumes and ranks them based on the relevance and quality of projects mentioned in their resumes.
The system helps recruiters and organizations automatically identify the most suitable candidates by evaluating their project experience.

Objective

To build an intelligent system that:
Analyzes resumes automatically
Extracts project information
Evaluates project relevance
Ranks candidates based on project quality
Reduces manual screening effort

Features

✅ Resume text extraction
✅ Project identification using NLP
✅ Semantic understanding using LLM
✅ Candidate scoring
✅ Automatic ranking
✅ Easy-to-use interface

Technologies Used
Python
Large Language Model (LLM)
NLP (Natural Language Processing)
Sentence Transformers / Ollama
Streamlit 
Pandas
Scikit-learn

How It Works
Step 1: Upload resumes
Step 2: System extracts text
Step 3: LLM analyzes projects
Step 4: Assigns score
Step 5: Ranks candidates
